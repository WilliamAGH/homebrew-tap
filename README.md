# WilliamAGH Homebrew Tap

Homebrew formulae for [WilliamAGH](https://github.com/WilliamAGH) projects.

## Installation

```bash
brew tap williamagh/tap
```

## Formulae

| Formula | Description | Install |
|---------|-------------|---------|
| [brief](https://github.com/WilliamAGH/brief) | Terminal-first chat client | `brew install williamagh/tap/brief` |

## Usage

**Stable release:**
```bash
brew install williamagh/tap/brief
```

**Nightly (latest from main):**
```bash
brew install --head williamagh/tap/brief
```

**Upgrade:**
```bash
brew upgrade brief                  # stable
brew upgrade --fetch-HEAD brief     # nightly
```
